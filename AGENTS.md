# Codex Project Instructions

This is the Vue/Vite site for the 弹药工程与爆炸技术导航站.

Before project work, read:

1. `CODEX_PROJECT_MEMORY.md`
2. `DATA_SCHEMA.md`
3. `README.md`
4. `research-drafts/knowledge-pool/00-宇少先看这个.md`
5. `research-drafts/knowledge-pool/README.md`
6. `../antigravity工作区/ANTIGRAVITY_KNOWLEDGE_POOL_WORKFLOW.md` when working with Antigravity

Key rule:

- Current priority is the knowledge pool: enterprise and university candidates in `research-drafts/knowledge-pool/`.
- Grok/Cherry is not the default workflow now.
- Antigravity or Codex can collect candidates directly.
- Knowledge-pool work does not edit `src/data/companies.ts`, `src/data/universities.ts`, pages, deployment, VPS, domain, or server files.
- Formal import happens later from strong candidates through clean JSON, dry-run, scoring, and data checks.

Use `npm run build` after data or code changes when practical.
