## 质量门槛

- Markdown 批次文件按类型保存到 `batches/mixed/`、`batches/enterprise/` 或 `batches/university/`。
- 推荐 raw JSON 放 `raw-json/`，文件名和批次名尽量一致。
- 目标是入库预备级：后续应能较快转成 clean JSON。
- 每条至少 1 个可打开关键入口，优先 2 个以上来源。
- 强候选建议至少 3 个来源，其中最好有官网 / 政府资料 / 学校就业网 / 招生网 / 招聘公告之一。
- 只有 1 个来源时，证据强弱最多写“中”，不能写“强”。
- 每个入选候选必须做多轮搜索，并记录检索关键词。
- 官网打不开、只有招聘平台或第三方转载时，证据强弱写“中 / 待核对”。
- 不写薪资、作息、待遇、分数线、招生人数、就业率。
- 不把研究生方向包装成本科专业。
- 不把待核对内容写成确定事实。
- 不合格条目可以写暂缓，不要硬凑。
